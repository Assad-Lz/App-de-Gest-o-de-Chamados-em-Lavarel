# Privacy Policy

This project, **Cellar Vinhos - Gestão de Chamados**, is an educational and study project.

## 1. Data Collection
As a local development project, this application does not collect or transmit any personal data to external servers, except for the database connection configured by the user (e.g., Supabase, PostgreSQL).

## 2. Local Processing
All data entered into the system (ticket titles, descriptions, names, emails) is stored locally or in the user's private database instance. No data is shared with the developers of this code.

## 3. Third-Party Services
The application may use:
- **Supabase/PostgreSQL**: For database storage.
- **Google Fonts**: For typography.
The privacy policies of these services apply when they are active.

## 4. Security
We have implemented several security measures (CSP, XSS protection, Backend Obfuscation) to ensure that the study environment is secure. However, as it is a study project, it should not be used for real production data without further audit.

## 5. Contact
For any questions regarding this study project, please contact the repository owner.
